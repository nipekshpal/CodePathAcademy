Loops in C

Module 5 · 18 min read · Intermediate

Loops are one of the most fundamental concepts in programming. They allow you to execute a block of code multiple times, which is essential for tasks like processing arrays, reading data, and implementing algorithms.

In C, there are three main types of loops: for, while, and do-while. Each has its own use case and syntax, and understanding when to use each is key to writing efficient code.

The for Loop

The for loop is the most commonly used loop in C. It's ideal when you know exactly how many times you want to iterate. 

// Print numbers 1 to 5
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

The while Loop

The while loop is used when you want to repeat a block of code as long as a condition is true. It's useful when you don't know the exact number of iterations beforehand. 

// Print numbers 1 to 5
int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    i++;
}

The do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the code block executes at least once before checking the condition. 

// Print numbers 1 to 5
int i = 1;
do {
    printf("%d\n", i);
    i++;
} while (i <= 5);

Key Differences

  • for loop: Best for known iteration counts, compact syntax.
  • while loop: Best when condition depends on external factors.
  • do-while loop: Best when you need at least one execution.
